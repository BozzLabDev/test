# Killer Snake Game

这是一个炫酷的贪吃蛇游戏，使用HTML、CSS和JavaScript实现。

## 功能特点

- 经典贪吃蛇玩法
- 霓虹风格界面和渐变背景
- 蛇头带有眼睛设计
- 食物具有发光和径向渐变效果
- 吃到食物时有粒子爆炸效果
- 随着分数增加，游戏速度逐渐提升
- 实时分数显示
- 游戏结束界面

## 如何游玩

使用键盘上的方向键（上、下、左、右）来控制蛇的移动方向。

## 部署到 GitHub Pages

要将此游戏部署到 GitHub Pages，请按照以下步骤操作：

1. 将此项目推送到 GitHub 仓库
2. 在 GitHub 仓库中，转到 Settings > Pages
3. 在 "Source" 部分选择 "Deploy from a branch"
4. 选择 "main" 或 "master" 分支，并点击 "Save"
5. 等待几分钟，您的游戏将在 `https://<your-username>.github.io/<repository-name>` 上可用

## 文件结构

- `index.html` - 主页面
- `style.css` - 样式表
- `script.js` - 游戏逻辑

## 自定义

您可以根据需要修改样式和游戏参数，如：
- 游戏速度
- 颜色主题
- 网格大小
- 粒子效果
